# Nightgauge Homebrew Tap

Homebrew formulae for the [Nightgauge](https://github.com/nightgauge/nightgauge) CLI.

The tap is reserved for Nightgauge's first public release. There is no published
formula yet; installation instructions will appear here after the release is
available and verified.

Formulae here are generated automatically by GoReleaser on each release. Do not edit by hand.
